📌 CodeAlpha_AgeCalculator

CodeAlpha Age Calculator is a simple and responsive web application that calculates a user's exact age in years, months, and days based on their date of birth.

This project was developed as part of the CodeAlpha Web Development Internship Program, using core web technologies like HTML, CSS, and JavaScript.

🚀 Features
📅 User-friendly input for Date of Birth
🧮 Accurate age calculation (Years, Months, Days)
⚡ Instant results using JavaScript Date object
📱 Responsive and clean UI design
❗ Input validation for better user experience
🛠️ Technologies Used
HTML – Structure of the web page
CSS – Styling and layout design
JavaScript – Age calculation logic
